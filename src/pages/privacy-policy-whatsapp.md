# Política de Privacidade - Sistema de Alarmes

**Última atualização:** Janeiro de 2025

## 1. Sobre Este Serviço

Este é um sistema privado de notificações de alarmes que utiliza a API do WhatsApp Business para enviar mensagens automáticas sobre eventos e alertas de infraestrutura pessoal.

## 2. Informações Coletadas

### 2.1 Dados que coletamos:
- **Número de telefone WhatsApp:** Necessário para envio das notificações
- **Nome do contato:** Para identificação do destinatário
- **Logs de mensagens:** Data, hora e status de entrega das notificações
- **Dados do alarme:** Tipo de alerta, origem e descrição do evento

### 2.2 Dados que NÃO coletamos:
- Conteúdo de conversas pessoais do WhatsApp
- Lista de contatos do seu WhatsApp
- Mídia ou arquivos pessoais
- Localização geográfica
- Dados bancários ou financeiros

## 3. Como Usamos as Informações

As informações coletadas são utilizadas exclusivamente para:
- Enviar notificações de alarmes e alertas do sistema
- Confirmar a entrega das mensagens
- Manter logs para auditoria e resolução de problemas
- Melhorar a confiabilidade do sistema de notificações

## 4. Compartilhamento de Dados

**Não compartilhamos, vendemos ou alugamos suas informações pessoais.** 

Os dados são processados através da:
- **API do WhatsApp Business (Meta):** Para envio das mensagens
- **Infraestrutura própria:** Onde os dados são armazenados e processados

## 5. Segurança dos Dados

Implementamos medidas de segurança para proteger suas informações:
- Criptografia de dados em trânsito (HTTPS/TLS)
- Acesso restrito aos sistemas
- Logs de acesso monitorados
- Autenticação segura para APIs

## 6. Retenção de Dados

- **Logs de mensagens:** Mantidos por 30 dias
- **Dados de contato:** Mantidos enquanto o serviço estiver ativo
- **Dados de alarmes:** Mantidos por 90 dias para análise

## 7. Seus Direitos

Você tem o direito de:
- Solicitar acesso aos seus dados pessoais
- Solicitar correção de dados incorretos
- Solicitar exclusão dos seus dados
- Cancelar o recebimento de notificações a qualquer momento

## 8. Base Legal

Este serviço opera com base no seu consentimento explícito ao cadastrar seu número para receber notificações.

## 9. Alterações nesta Política

Podemos atualizar esta política periodicamente. Notificaremos sobre mudanças significativas através do sistema de notificações.

## 10. Contato

Para questões sobre privacidade ou exercer seus direitos, entre em contato:
- **Email:** gabriel@prandogabriel.dev
- **WhatsApp:** Responda STOP para cancelar notificações

## 11. Conformidade

Este serviço está em conformidade com:
- Lei Geral de Proteção de Dados (LGPD) - Brasil
- Políticas da API do WhatsApp Business da Meta
- Boas práticas de privacidade e segurança de dados

## 12. Consentimento

Ao utilizar este serviço de notificações, você concorda com esta política de privacidade e com o processamento dos seus dados conforme descrito acima.
